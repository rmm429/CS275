To run rmm429_HW1.html, double-click on the file, which will open the HTML page in your default browser.
To create the table of Fibonacci values, enter an integer into the text box and click the button "Calculate."
NOTE: The input has to be an integer greater than or equal to 0.  Otherwise, an error message will appear.
Once the "Calculate" button is pressed, a table will appear with all of the Fibonacci values and their respective indexes.
